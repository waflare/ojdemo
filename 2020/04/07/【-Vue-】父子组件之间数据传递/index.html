父子组件之间数据传递

一、父组件向子组件传递数据

<div id="app">
    <counter :number='count'></counter>
</div>

<script type="text/javascript">
    // 子组件
    var counter = {
        template: '<div>{{ number }}</div>',
        // 使用 props 接收父组件传过来的参数
        props: [
            'number'
        ]
    };

    var vm = new Vue({
        el: '#app',
        components: {
            counter,
        },
        data() {
            return {
                count: 0
            }
        }
    })
</script>

注意：

在 Vue 里存在一个叫做单向数据流的概念，也就是父组件可以随意的向子组件传递参数，子组件接收到父组件的参数后，只能使用，不能做修改，因为父组件传过来的可能是引用型数据，子组件乱改的话很可能影响其他组件使用这个数据。例如：

var counter = {
    template: '<div @click="handleClick">{{ number }}</div>',
    props: [
        'number'
    ],
    methods: {
        handleClick() {
            this.number ++;
        }
    }
};

这里子组件把父组件传过来的number参数做了加一修改，运行看似是有效果的，点击数字可以实现加一，但是打开控制台就可以看到有报错。

解决方法——子组件 clone 一份数据，对自己的数据进行修改就可以了

var counter = {
    template: '<div @click="handleClick">{{ count }}</div>',
    props: [
        'number'
    ],
    data() {
        return {
            count: this.number
        }
    },
    methods: {
        handleClick() {
            this.count ++;
        }
    }
};

二、子组件数据变更后向父组件传递数据

子组件向父组件传值我们用事件来实现，子组件通过 this.$emit 向外触发事件，实现传递数据

<div id="app">
    <!-- 这里的 @numberInc 对应子组件里 $emit 里设置的事件名称 -->
    <counter :number='count' @number-inc='changeNumber'></counter>
    父组件的 count 值： {{ count }}
</div>

<script type="text/javascript">
    var counter = {
        template: '<div @click="handleClick">{{ count }}</div>',
        props: [
            'number'
        ],
        data() {
            return {
                count: this.number
            }
        },
        methods: {
            handleClick() {
                this.count ++;
                /*
                * numberInc 事件名可以随意修改(自定义)但是不要用大写
                * 因为在这里大小写不敏感，后面的参数个数也可以添加多个
                */
                this.$emit("number-inc", this.count);
            }
        }
    };

    var vm = new Vue({
        el: '#app',
        components: {
            counter,
        },
        data() {
            return {
                count: 0
            }
        },
        methods: {
            // 这里的参数名 counter 也是随便设的
            changeNumber(counter) {
                this.count = counter;
            }
        }
    })
</script>
